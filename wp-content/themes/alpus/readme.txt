Alpus is fast, fully customizable WordPress theme suitable for blog, personal portfolio, business and eCommerce Websites.

== Description ==
Alpus is a versatile and modern multipurpose WordPress theme that can be used for a wide range of websites, including business, portfolio, blog, e-commerce, and more. It features a clean and elegant design, with a focus on usability and user experience.

The theme comes with a wide range of customization options, allowing you to easily modify the layout, colors, fonts, and other elements to suit your specific needs. You can choose from multiple pre-built demos, each with its own unique style and layout, or create your own custom design using the intuitive drag-and-drop page builder.

Alpus is fully responsive and optimized for performance, ensuring that your website looks great and loads quickly on all devices, including desktops, laptops, tablets, and smartphones. It also includes support for popular plugins like WooCommerce, Contact Form 7, and Yoast SEO, making it easy to extend the functionality of your website.

Other features of Alpus include a powerful theme options panel, multiple header and footer styles, custom widgets, social media integration, Google Fonts integration, and much more. With its flexibility and ease of use, Alpus is an excellent choice for anyone looking to create a professional and engaging website.

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "Alpus"
3. Install and activate
